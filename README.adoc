== Examples input

* Precedence
+
[cols="1,1,1"]
|===
|Input |Correct |Error

|stem:[x + 5 * y]
|stem:[x + (5 * y)]
|stem:[(x + 5) * y]
|===

* Associativity
+
[cols="1,1,1"]
|===
|Input |Correct |Error

|stem:[7 - 3 + 2]
|stem:[(7 - 3) + 2]
|stem:[7 - (3 + 2)]
|===

== Grammar for expressions

E:: Expression
T:: Term
F:: Factor

=== Variations

* stem:[E -> T + E | T - E | T]
* stem:[T -> F * T | F / T | F]
* stem:[F -> ID | "Integers" | (E) - F]
